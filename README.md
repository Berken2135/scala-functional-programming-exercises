# Scala Functional Programming Exercises

This repository contains a collection of basic Scala exercises designed to demonstrate
fundamental programming and functional programming concepts.

## Topics Covered
- List processing
- Recursion
- Pattern matching
- For-comprehensions
- Basic functional thinking in Scala

## Technologies
- Scala 3

## How to Run
Each exercise includes a `@main` method.
You can run them individually using:

```bash
scala Main.scala
```
